
 
# Visit Hyderabad

A clean, responsive, multi-page website that highlights Hyderabad's heritage, culture, cuisine, and iconic attractions.



## Overview

Visit Hyderabad is a static website showcasing key destinations such as Charminar, Golconda Fort, Ramoji Film City, Birla Mandir, and other popular places. The layout is designed to be readable and consistent across desktop and mobile screens.

## Features

- Responsive layout with a consistent grid for content sections
- Dedicated pages for major attractions
- Call-to-action section and visit inquiry form
- Lightweight and fast to load

## Built With

- HTML5
- CSS3

## Pages

- Home: index.html
- Charminar: Charminar.html
- Golconda Fort: GolcondaFort.html
- Ramoji Film City: RamojiFilmCity.html
- Birla Mandir: BirlaMandir.html
- Other Places: Otherplaces.html
- Visit Form: Visit.html

## How to Run

1. Download or clone the repository.
2. Open index.html in your browser.
